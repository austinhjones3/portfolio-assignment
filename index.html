Projects

Get-Coins

I decided the best way to review my knowledge of making AJAX requests with async-await in Node.js would be to make a small application that calls an API I’ve never used before. I chose Coinbase’s non-authenticated public API options. The users have four options for requests. When a request is selected, it is fulfilled with Axios. Then the user has the choice to make another request or to end the run. The front end for this project is the old reliable terminal.

Decoder Ring

A few other engineering students and I at Thinkful worked together to create our solutions for the Caesar cipher, the substitution cipher, and the Polybius square cipher in Node.js. We also built mocha and chai tests for our solutions. It was fun to work through these ciphers using more primitive programming practices. It helped me to nail down some of the basics. Reviewing the basics is important because I am also learning more and more modern syntax.

Local Library

The local library is a back-end Node.js data handling and structuring program that uses many array methods and a ton of ES6 syntax. Sets of data such as authors, books, and library account members interact with each other. The goal was to organize the data into different filter sets that show the data in a more palatable interface. Scrolling through lines and lines of data does not make for the best user experience. Here, I learned a lot about arrays and objects.
